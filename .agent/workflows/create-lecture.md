# Робочий процес створення лекції

Цей проект використовує Reveal.js через `reveal-md` з кастомним скриптом-ранером.

## 1. Створення контенту
// turbo
1. Створіть нову директорію: `mkdir -p slides/lecture-XX`
2. Створіть файл лекції: `slides/lecture-XX/lecture-XX.md`
3. Використовуйте шаблон з `AGENTS.md` (титульний слайд, frontmatter без `theme`, нотатки спікера).

## 2. Перегляд та розробка
// turbo
1. Запустіть режим розробки: `npm run dev -- lecture-XX`
2. Відкрийте http://localhost:1950

## 3. Генерація PDF та Експорт (Нагадування)
1. Нагадайте користувачу, що після завершення роботи над лекцією (коли сесія буде закінчена), він може експортувати результати за допомогою команд:
   - PDF: `npm run print -- lecture-XX`
   - Статичний HTML: `npm run export -- lecture-XX`

> **Важливо:** Завжди дотримуйтесь "Золотого стандарту" слайда з `AGENTS.md` для забезпечення якості контенту та коректної роботи нотаток спікера.
