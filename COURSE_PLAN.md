## Лекція 1: Вступ в Frontend. Інструменти веб-розробника
*Лабораторна робота №1*

- Що таке Frontend: HTML — структура, CSS — стиль, JS — поведінка
- **Розрізнення ролей:** Frontend ≠ Дизайн ≠ Backend; frontend як міст між дизайном та бізнес-логікою
- **HTTP-цикл:** як браузер спілкується з сервером (Request → Server → DB → Response → Render)
- **Браузери:** Chrome, Firefox, Safari, Edge; принцип «розробляємо у Chrome, тестуємо у всіх»
- **Chrome DevTools:** Elements, Console, Network, Application, Performance, Device Toolbar
- `console.log` — перший інструмент дебагінгу
- **Редактори коду:** VS Code (стандарт), WebStorm, Sublime Text
- **Плагіни VS Code:** ESLint, Prettier, Live Server, Auto Rename Tag, CSS Peek, GitLens
- **Клавіатурні скорочення VS Code:** Command Palette, мультикурсор, переміщення рядків, термінал
- **Emmet:** абревіатури для швидкої генерації HTML (`!`, `div>ul>li*3`, BEM-компоненти)
- **Git:** машина часу для коду — init, add, commit, push, branches
- **Git vs GitHub:** локальна утиліта vs хмарний сервіс; GitHub як портфоліо розробника
- **Реальний робочий день:** Jira/Trello → git branch → код у VS Code → DevTools → Pull Request → Merge → Deploy
- Онлайн-пісочниці: CodeSandbox

## Лекція 2: Основи HTML. Структура документа
*Лабораторна робота №2*

- Що таке HTML; коротка історія
- Структура документа: `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`
- Метадані: `<meta charset>`, `<meta viewport>`, `<title>`
- Блокові та рядкові елементи
- Текст: `<h1>`–`<h6>`, `<p>`, `<strong>`, `<em>`, `<br>`, `<hr>`
- Містки (контейнери): `<div>`, `<span>`
- Списки: `<ul>`, `<ol>`, `<li>`
- Гіперпосилання: `<a>` — абсолютні та відносні шляхи, `target`
- Зображення: `<img>` — `src`, `alt`, формати

## Лекція 3: Семантична розмітка HTML5
*Лабораторна робота №3*

- Навіщо потрібна семантика: SEO, доступність (a11y), читаність коду
- Семантичні теги структури: `<header>`, `<nav>`, `<main>`, `<footer>`
- Теги контенту: `<article>`, `<section>`, `<aside>`
- Типографіка: `<figure>`, `<figcaption>`, `<time>`, `<address>`, `<blockquote>`
- Різниця між `<div>` і семантичними тегами
- Як accessibility-дерево зчитується скрінрідерами

## Лекція 4: Таблиці в HTML
*Лабораторна робота №4*

- Призначення таблиць: тільки для табличних даних
- Базова структура: `<table>`, `<tr>`, `<td>`, `<th>`
- Семантика таблиці: `<thead>`, `<tbody>`, `<tfoot>`, `<caption>`
- Об'єднання комірок: `colspan` та `rowspan`
- Базова стилізація таблиць через CSS
- Доступність: атрибут `scope`

## Лекція 5: HTML Форми
*Лабораторна робота №5*

- Призначення та робота форм: `<form>`, атрибути `action`, `method`
- Елементи вводу `<input>`: типи `text`, `email`, `password`, `date`, `radio`, `checkbox`, `file`, `range`
- Інші елементи: `<textarea>`, `<select>` + `<option>`, `<button>`
- Підписи: `<label>` та атрибут `for`; `<fieldset>` + `<legend>`
- Клієнтська валідація: `required`, `pattern`, `min`, `max`, `minlength`
- Псевдокласи форм: `:valid`, `:invalid`, `:focus`

---

## Лекція 6: Основи CSS: селектори та стилізація
*Лабораторна робота №6*

- Способи підключення CSS: inline, internal (`<style>`), external (`<link>`)
- Синтаксис: `selector { property: value; }`
- Типи селекторів: тег, клас (`.`), ID (`#`), атрибут (`[]`), універсальний (`*`)
- Комбінатори: нащадок (` `), дочірній (`>`), сусід (`~`, `+`)
- Псевдокласи: `:hover`, `:active`, `:focus`, `:nth-child`, `:first-child`
- Псевдоелементи: `::before`, `::after`
- Специфічність і каскад; `!important`
- Боксова модель: `margin`, `border`, `padding`, `content`; `box-sizing`
- Властивість `display`: `block`, `inline`, `inline-block`, `none`
- CSS-змінні: `--variable`

---

## Лекція 7: CSS Layout — Flexbox та Grid
*Лабораторна робота №7*

- **Flexbox** — одновимірний макет:
  - `display: flex`, головна та перехресна осі
  - `justify-content`, `align-items`, `align-self`
  - `flex-wrap`, `flex-grow`, `flex-shrink`, `flex-basis`
  - `gap`, `order`
- **CSS Grid** — двовимірний макет:
  - `display: grid`, `grid-template-columns`, `grid-template-rows`
  - `grid-gap` / `gap`, `grid-area`
  - `fr`, `minmax()`, `repeat()`
- Коли використовувати Flexbox, а коли Grid
- Типові паттерни: navbar, картки, сітка сторінки

## Лекція 8: Робота з макетом (Figma) та Адаптивна верстка
*Лабораторна робота №8*

- **Figma:** огляд інтерфейсу, Inspect-панель, експорт ресурсів
- Аналіз відступів, шрифтів та кольорів з макету
- **Адаптивна верстка:** що таке Responsive Design
- `<meta name="viewport">` — чому це обов'язково
- Media Queries: синтаксис `@media`, розміри breakpoints
- Підходи: Mobile First vs Desktop First
- Адаптивні зображення: `max-width: 100%`
- Відносні одиниці: `%`, `em`, `rem`, `vw`, `vh`

## Лекція 10: Основи JavaScript: синтаксис та базові алгоритми
*Лабораторна робота №10*

- Що таке JavaScript; місце у веб-стеку
- Підключення скриптів: `<script>`, атрибут `defer`
- Змінні: `var`, `let`, `const`
- Типи даних: number, string, boolean, null, undefined, object
- Оператори, умови: `if/else`, тернарний, `switch`
- Цикли: `for`, `while`, `for...of`, `for...in`
- Функції: declaration, expression, arrow function
- Масиви: `push`, `pop`, `map`, `filter`, `find`, `reduce`
- Об'єкти: створення, доступ до властивостей, деструктуризація
- `console.log`, базове налагодження

## Лекція 11: JavaScript в браузері (DOM) та Асинхронність
*Лабораторна робота №11*

- **DOM API:** що таке Document Object Model
- Пошук елементів: `querySelector`, `querySelectorAll`, `getElementById`
- Зміна вмісту: `textContent`, `innerHTML`, `classList`, `style`
- Створення DOM-елементів: `createElement`, `append`, `remove`
- **Події:** `addEventListener`, типи подій (click, input, submit, keydown)
- Делегування подій
- **localStorage:** `setItem`, `getItem`, `removeItem`
- **Асинхронність:** Event Loop, `setTimeout`, `Promise`, `async/await`
- **Fetch API:** GET-запит, обробка JSON, catch помилок

## Лекція 12: Вступ до React та Деплой
*Лабораторна робота №12*

- Чому React? Virtual DOM, компонентний підхід
- JSX — синтаксис, вбудовані вирази, умовний рендеринг
- Компоненти: функціональні компоненти, іменування
- Props — передача даних від батька до дочірнього компонента
- State: хук `useState` — ініціалізація, оновлення
- Рендеринг списків: `map()` та атрибут `key`
- Створення проекту: `npm create vite@latest`
- **Деплой:** GitHub Pages (`gh-pages`), Vercel — покроковий процес
- Огляд курсу та наступні кроки у розвитку
